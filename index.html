Getting Started with Leading AI APIs

Need to prototype an AI assistant for your project? Follow the quick-start checklists below to authenticate with OpenAI’s ChatGPT API and Anthropic’s Claude API using their official Python SDKs.

1. ChatGPT (OpenAI API)

  1. Create an OpenAI account at platform.openai.com/signup.
  2. Generate an API key from the API Keys page in your dashboard.
  3. Install the OpenAI Python SDK: pip install openai.
  4. Use the sample below to send your first chat completion:
import openai

openai.api_key = "your-api-key-here"

response = openai.ChatCompletion.create(
    model="gpt-3.5-turbo",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Hello, who are you?"}
    ]
)

print(response["choices"][0]["message"]["content"])

2. Claude (Anthropic API)

  1. Request access at console.anthropic.com.
  2. After approval, create an API key from your dashboard.
  3. Install the Anthropic Python SDK: pip install anthropic.
  4. Send a message with the following starter code:
import anthropic

client = anthropic.Anthropic(api_key="your-api-key-here")

response = client.messages.create(
    model="claude-2.1",
    max_tokens=128,
    messages=[
        {"role": "user", "content": "Hello, who are you?"}
    ]
)

print(response.content)

Tip: Both platforms support HTTPS requests, streaming responses, and multiple models—replace "your-api-key-here" with your live secret key before deploying.
